# 系统配置：前端控件类型与 options 参数

> 适用于 `web/src/views/system/config/index.vue` 当前实现。后续新增类型请在此补充。

## 可用控件类型
当前支持：`text`、`textarea`、`number`、`switch`、`select`、`radio`、`checkbox`、`color`、`object`、`array`、`array-json`（上传 upload 预留，尚未实现）。

## 通用约定
- `options` 字段存 JSON（字符串或对象均可），未配置时为空。
- 未识别的字段会被忽略；未识别的控件类型会退回为普通输入框。
- `remark` 作为占位符（placeholder）。
- 值（value）全部以字符串存库；数组/对象使用 JSON 字符串。

## 各类型参数

### text / textarea
- `placeholder`: string
- textarea 额外：使用 autosize，行数不可配（需改造再扩展）。

### number
- `min`: number 可选
- `max`: number 可选
- `step`: number 可选（默认 1）

### switch
- 无需 options；值固定为 `"1"` / `"0"`。

### select
- `options`: Array<{ label?: string; value: string | number }>
- `multiple`: boolean （true 则多选，值存为 JSON 数组字符串）
- `placeholder`: string 可选

### radio
- `options`: Array<{ label?: string; value: string | number }>

### checkbox
- `options`: Array<{ label?: string; value: string | number }>
- 值为 JSON 数组字符串，例如 `["a","b"]`。

### color
- 无需 options；显示一个输入框 + 取色器，值为 hex（含 alpha）。

### object
- 纯 JSON 编辑区域；无 options；值需为对象 JSON 字符串。

### array
支持两种模式：
- **列表模式**：`options.fields` 存在且为数组，按字段渲染行。
- **JSON 模式**：`options.fields` 为空或值为非对象数组/多维数组时自动切换；也可手动“切换 JSON 阅读”。

`options.fields` 结构：
```json
[
  {
    "key": "domain",
    "label": "备用域名",
    "type": "input",          // 可选：input(默认)/textarea/number/switch/color/radio/checkbox/select
    "placeholder": "https://a.static.example.com",
    "min": 0,                 // 仅 number
    "max": 100,               // 仅 number
    "step": 1,                // 仅 number
    "multiple": true,         // select/checkbox 多选
    "options": [              // radio/checkbox/select 使用
      { "label": "国内", "value": "cn" },
      { "label": "海外", "value": "us" }
    ]
  }
]
```
- 列表模式值示例（存为字符串）：`[{"domain":"https://a.static.example.com"}]`
- JSON 模式值：任意数组结构，前端不做结构化渲染，仅提供 JSON 编辑。
- 排序：列表模式默认支持拖拽排序；如不需要排序，在 `options` 中设置 `"sortable": false` 关闭拖拽句柄。

### array-json
- 强制 JSON 编辑模式（不尝试结构化）。
- 值示例：`["10.0.0.1","10.0.0.2"]`

### editor（富文本，WangEditor 封装）
- 组件：`ArtWangEditor`，值为 HTML 字符串。
- options：
  - `placeholder`: string（默认“请输入内容”）
  - `height`: string（如 `"360px"`，默认 360px）
  - `mode`: `"default"` | `"simple"`（默认 `"default"`）
  - `toolbarKeys`: string[] 自定义工具栏
  - `insertKeys`: { index: number; keys: string[] } 插入工具
  - `excludeKeys`: string[] 排除工具
  - `uploadConfig`: { `server`?: string; `maxFileSize`?: number; `maxNumberOfFiles`?: number; `fieldName`?: string; `allowedFileTypes`?: string[] }
- 上传默认指向后端 `/api/common/upload/wangeditor`，带授权头；如需自定义请覆盖 `uploadConfig.server`。

## 典型 options 示例

- 单选下拉：
```json
{
  "options": [
    { "label": "是", "value": "1" },
    { "label": "否", "value": "0" }
  ]
}
```

- 多选下拉：
```json
{
  "multiple": true,
  "options": [
    { "label": "华东", "value": "cn-east" },
    { "label": "华北", "value": "cn-north" }
  ]
}
```

- 一维数组（列表模式）：
```json
{
  "fields": [
    { "key": "domain", "label": "备用域名", "type": "input", "placeholder": "https://a.static.example.com" }
  ]
}
```

- 对象数组（多列列表模式）：
```json
{
  "fields": [
    { "key": "key", "label": "Key", "type": "input" },
    { "key": "value", "label": "值", "type": "textarea", "placeholder": "多行内容" }
  ]
}
```

- 纯 JSON 数组（无需 fields）：
```json
{
  "note": "不提供 fields 时，自动进入 JSON 模式"
}
```

## 值格式速查
- text / textarea / number / select / radio / color / object：字符串（其中 object 为 JSON 对象字符串）。
- switch：`"1"` / `"0"`。
- checkbox：JSON 数组字符串。
- array：列表模式存对象数组字符串；JSON 模式存任意数组字符串。
- array-json：任意数组字符串。
