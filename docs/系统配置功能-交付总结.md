## 🚀 快速开始

### 1. 执行数据库初始化

```bash
# 执行 SQL 脚本
mysql -u root -p your_database < docs/config_group_init.sql
```

或者在数据库管理工具（如 phpMyAdmin）中执行：
```sql
UPDATE `xy_sys_config` 
SET 
  `value` = '[{"group":"basics","groupName":"基础设置","icon":"ri:settings-3-line","description":"系统常规信息配置","sort":0}...]',
  `options` = '{"fields":[...]}',
  `update_time` = UNIX_TIMESTAMP()
WHERE `key` = 'config_group';
```

### 2. 启动服务

```bash
# 后端服务
cd server
gf run main.go

# 前端服务
cd newweb
pnpm run dev
```

### 3. 访问配置页面

```
http://localhost:3006/#/system/config
```

## 💡 使用示例

### 前端使用

```vue
<template>
  <!-- 颜色选择器 -->
  <ArtColorPicker v-model="themeColor" />
  
  <!-- 图标选择器 -->
  <ArtIconSelector v-model="menuIcon" />
  
  <!-- 单图上传 -->
  <ArtImageUpload v-model="logoUrl" :max-size="5" />
  
  <!-- 多图上传 -->
  <ArtImageUpload v-model="carouselImages" multiple :limit="9" />
  
  <!-- 单文件上传 -->
  <ArtFileUpload v-model="contractFile" accept=".pdf,.doc" />
  
  <!-- 多文件上传 -->
  <ArtFileUpload v-model="attachments" multiple :limit="10" />
</template>
```

### 后端使用

```go
// 获取配置
cfg, err := service.SysConfig().GetConfigByGroup(ctx, "basics")
if err != nil {
    return err
}

// 使用配置项
siteName := cfg["site_name"]        // 站点名称
themeColor := cfg["theme_color"]    // 主题颜色
logoUrl := cfg["site_logo"]         // LOGO地址
isClosed := cfg["site_closed"] == "1" // 是否关闭
```

## 🔐 权限控制

### allow_del 字段

- `0` - 核心配置，不允许删除（菜单项禁用）
- `1` - 可选配置，允许删除

```sql
-- 设置核心配置
UPDATE xy_sys_config SET allow_del = 0 WHERE `key` = 'site_name';

-- 设置可删除配置
UPDATE xy_sys_config SET allow_del = 1 WHERE `key` = 'custom_field';
```

## 📞 技术支持

### 问题排查

1. **配置加载失败**
   - 检查数据库中是否有 `config_group` 记录
   - 检查后端接口是否正常

2. **组件显示异常**
   - 检查 configFormData 是否有值
   - 检查 schema 是否加载完成

3. **保存失败**
   - 检查字段类型是否正确
   - 检查后端日志
