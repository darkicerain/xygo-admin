## 📦 新增的自定义组件

### 1. ArtColorPicker - 颜色选择器
**位置**: `newweb/src/components/core/forms/art-color-picker/index.vue`

**特性**:
- Element Plus 颜色选择器 + 输入框组合
- 15种预定义常用颜色
- 支持手动输入颜色值
- 实时颜色预览
- 支持 hex、rgb、hsl、hsv 格式

**Props**:
- `modelValue` - 颜色值
- `placeholder` - 占位文本
- `showAlpha` - 是否显示透明度
- `colorFormat` - 颜色格式

---

### 2. ArtImageUpload - 图片上传组件
**位置**: `newweb/src/components/core/forms/art-image-upload/index.vue`

**特性**:
- 单图/多图上传模式
- 拖拽排序（多图模式）
- 图片预览（点击放大查看）
- 悬停显示操作按钮
- 文件大小限制
- 支持 JPG、PNG、GIF、WebP

**Props**:
- `modelValue` - 图片URL（单图）或逗号分隔的URL（多图）
- `multiple` - 是否多图模式
- `limit` - 最大上传数量（默认9）
- `maxSize` - 最大文件大小MB（默认5）
- `accept` - 允许的文件类型
- `uploadAction` - 上传接口地址

**多图模式**:
- 支持拖拽调整顺序
- 120x120 网格展示
- 最多支持指定数量的图片

---

### 3. ArtFileUpload - 文件上传组件
**位置**: `newweb/src/components/core/forms/art-file-upload/index.vue`

**特性**:
- 单文件/多文件上传模式
- 智能文件图标识别（PDF、Word、Excel、ZIP等）
- 文件大小自动格式化
- 下载/删除功能
- 文件类型限制
- 支持所有文件格式

**Props**:
- `modelValue` - 文件URL
- `multiple` - 是否多文件模式
- `limit` - 最大上传数量（默认10）
- `maxSize` - 最大文件大小MB（默认10）
- `accept` - 允许的文件类型（默认"*"）
- `uploadAction` - 上传接口地址

**文件图标映射**:
- PDF → ri:file-pdf-line
- Word → ri:file-word-line
- Excel → ri:file-excel-line
- ZIP → ri:file-zip-line
- Code → ri:file-code-line
- Video → ri:file-video-line
- Audio → ri:file-music-line
- 其他 → ri:file-line

## 🔧 组件使用示例

### 基础输入类

```json
// 文本输入
{"type": "text", "key": "site_name", "name": "站点名称", "value": "XY-GO"}

// 密码输入
{"type": "password", "key": "smtp_pass", "name": "SMTP密码", "value": ""}

// 多行文本
{"type": "textarea", "key": "site_desc", "name": "站点描述", "value": "", "options": {"rows": 4}}

// 数字输入
{"type": "number", "key": "smtp_port", "name": "SMTP端口", "value": "465", "options": {"min": 1, "max": 65535}}
```

### 选择类

```json
// 开关
{"type": "switch", "key": "site_closed", "name": "站点关闭", "value": "0"}

// 下拉选择
{
  "type": "select",
  "key": "protocol",
  "name": "协议",
  "value": "http",
  "options": {
    "options": [
      {"label": "HTTP", "value": "http"},
      {"label": "HTTPS", "value": "https"}
    ]
  }
}

// 下拉多选
{
  "type": "selects",
  "key": "allowed_ips",
  "name": "允许IP",
  "value": [],
  "options": {
    "options": [
      {"label": "本地", "value": "127.0.0.1"},
      {"label": "内网A", "value": "192.168.1.1"}
    ]
  }
}
```

### 日期时间类

```json
// 日期时间
{"type": "datetime", "key": "publish_time", "name": "发布时间", "value": "2026-01-15 12:00:00"}

// 日期
{"type": "date", "key": "start_date", "name": "开始日期", "value": "2026-01-15"}

// 年份
{"type": "year", "key": "found_year", "name": "成立年份", "value": "2026"}

// 时间
{"type": "time", "key": "work_time", "name": "工作时间", "value": "09:00:00"}
```

### 特殊类

```json
// 颜色选择器
{"type": "color", "key": "theme_color", "name": "主题颜色", "value": "#409EFF"}

// 图标选择器
{"type": "icon", "key": "menu_icon", "name": "菜单图标", "value": "ri:home-line"}

// 富文本编辑器
{"type": "editor", "key": "site_notice", "name": "站点公告", "value": "<p>欢迎使用</p>"}

// 城市选择器
{"type": "city", "key": "company_city", "name": "公司所在地", "value": []}
```

### 文件上传类

```json
// 单图上传
{
  "type": "image",
  "key": "site_logo",
  "name": "网站LOGO",
  "value": "",
  "options": {"maxSize": 5}
}

// 多图上传
{
  "type": "images",
  "key": "carousel",
  "name": "轮播图",
  "value": "",
  "options": {"limit": 9, "maxSize": 5}
}

// 单文件上传
{
  "type": "file",
  "key": "contract",
  "name": "合同文件",
  "value": "",
  "options": {"accept": ".pdf,.doc,.docx", "maxSize": 10}
}

// 多文件上传
{
  "type": "files",
  "key": "attachments",
  "name": "附件",
  "value": "",
  "options": {"limit": 10, "maxSize": 10}
}
```

## 🛠️ 已修复的问题

### 1. ✅ vuedraggable 导入错误
**问题**: 使用了 `vuedraggable` 但项目使用的是 `vue-draggable-plus`  
**修复**: 改用 `VueDraggable` 组件并调整模板语法

### 2. ✅ 类型检查错误
**问题**: switch/checkbox 比较时类型不匹配  
**修复**: 统一转换为字符串后再比较

### 3. ✅ 数字类型转换
**问题**: ElInputNumber 期望 Number 但接收到 String  
**修复**: 在数据加载时自动转换类型

## 💻 后端调用示例

```go
// 获取配置
cfg, err := service.SysConfig().GetConfigByGroup(ctx, "basics")
if err != nil {
    return err
}

// 使用不同类型的配置
siteName := cfg["site_name"]                    // text - 字符串
smtpPort := cfg["smtp_port"]                    // number - 需要转换
isClosed := cfg["site_closed"] == "1"           // switch - 布尔判断
themeColor := cfg["theme_color"]                // color - 颜色值
logoUrl := cfg["site_logo"]                     // image - 图片URL
images := strings.Split(cfg["carousel"], ",")   // images - 多图URL数组
```
