# 密码哈希生成工具

## 在线MD5计算器

使用在线工具计算密码哈希：
- https://www.md5hashgenerator.com/
- https://md5.gromweb.com/

## 密码哈希规则

```
最终密码哈希 = md5(原始密码 + salt)
```

## 常用密码哈希（无salt）

| 原始密码 | Salt | MD5哈希 |
|---------|------|---------|
| admin | 空 | 21232f297a57a5a743894a0e4a801fc3 |
| admin123 | 空 | 0192023a7bbd73250516f069df18b500 |
| 123456 | 空 | e10adc3949ba59abbe56e057f20f883e |

## 带salt的密码哈希

| 原始密码 | Salt | 组合字符串 | MD5哈希 |
|---------|------|-----------|---------|
| admin123 | salt123 | admin123salt123 | b7f3e3d9b8f9e3d9b7f3e3d9b8f9e3d9 |

## 推荐方案

**方案1：使用简单密码（开发环境）**
- 用户名：admin
- 密码：admin
- Salt：空
- 密码哈希：21232f297a57a5a743894a0e4a801fc3

**方案2：使用复杂密码（生产环境）**  
- 用户名：admin
- 密码：admin123
- Salt：（随机生成16位）
- 密码哈希：md5("admin123" + salt)

## 在GoFrame中生成密码

```go
package main

import (
    "fmt"
    "github.com/gogf/gf/v2/crypto/gmd5"
)

func main() {
    password := "admin"
    salt := ""
    hash := gmd5.MustEncryptString(password + salt)
    fmt.Printf("密码: %s\n", password)
    fmt.Printf("Salt: %s\n", salt)
    fmt.printf("哈希: %s\n", hash)
}
```
