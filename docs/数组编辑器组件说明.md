# ArtArrayEditor 多维数组编辑器 - 完整功能说明

## 🎯 组件概述

`ArtArrayEditor` 是一个功能强大的多维数组编辑器组件，专为复杂的数据配置场景设计。

### 核心特性

- ✅ **20+种字段类型** - 覆盖所有常见输入场景
- ✅ **拖拽排序** - 支持调整数据顺序
- ✅ **动态增删** - 实时添加/删除数据行
- ✅ **响应式布局** - 自适应网格布局
- ✅ **类型安全** - 完整的 TypeScript 支持
- ✅ **Art 风格** - 统一的视觉设计

## 🎬 组件交互

### 1. 添加数据
- 点击"添加数据"按钮
- 自动创建新行，所有字段填充默认值
- 新行自动滚动到视图中

### 2. 删除数据
- 点击行右侧的删除按钮
- 弹出二次确认对话框
- 确认后删除该行数据

### 3. 拖拽排序
- 鼠标悬停显示拖拽手柄
- 拖动手柄可调整顺序
- 松开鼠标后数据自动更新

### 4. 编辑数据
- 直接在字段中输入/选择
- 实时更新到 v-model
- 支持各种输入组件

### 5. 清空全部
- 点击"清空全部"按钮
- 显示数据数量确认
- 确认后清空所有数据

## 🎨 视觉设计

### 布局结构
```
┌──────────────────────────────────────┐
│ 数据列表 (3项)    [添加数据] [清空]   │ ← 头部
├──────────────────────────────────────┤
│ [≡] 1  [姓名] [角色] [标签] [激活] [×]│ ← 第1行
│ [≡] 2  [姓名] [角色] [标签] [激活] [×]│ ← 第2行  
│ [≡] 3  [姓名] [角色] [标签] [激活] [×]│ ← 第3行
└──────────────────────────────────────┘
  ↑   ↑    ↑字段区域（响应式网格）  ↑
  拖拽 序号                        删除
```

### 视觉特点
- 卡片式布局，圆角边框
- 悬停显示拖拽手柄
- 悬停高亮整行
- 序号圆角徽章（主题色）
- 响应式网格（自适应列数）

### 配色方案
- 边框：var(--art-gray-300)
- 背景：var(--default-box-color)
- 悬停：var(--theme-color) 边框

## 🔧 Props API

| 参数 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| modelValue | 数组数据 | any[] | [] |
| fields | 字段配置 | FieldConfig[] | [] |
| showIndex | 显示序号 | boolean | true |
| sortable | 支持拖拽排序 | boolean | true |
| groupable | 支持分组 | boolean | false |

## 🎯 FieldConfig API

```typescript
interface FieldConfig {
  key: string           // 字段键名（必填）
  label: string         // 字段标签（必填）
  type: string          // 字段类型（必填，见下表）
  
  // 通用配置
  placeholder?: string  // 占位文本
  default?: any         // 默认值
  
  // 数字类型
  min?: number          // 最小值
  max?: number          // 最大值
  
  // 选择类型
  options?: Array<{     // 选项列表
    label: string
    value: any
  }>
  
  // 文件上传类型
  maxSize?: number      // 文件大小限制（MB）
  limit?: number        // 上传数量限制
  accept?: string       // 文件类型限制
  
  // 多行文本
  rows?: number         // 行数
}
```
